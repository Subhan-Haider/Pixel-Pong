# Security Policy

## Supported Versions

The following versions of **Pixel Pong** are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

We take the security of our project seriously. If you find a security vulnerability, please do not report it through public issues. Instead:

1.  **Contact us:** Please reach out to Subhan Haider directly via GitHub.
2.  **Wait for a response:** We will acknowledge your report within 48 hours.
3.  **Coordinated Disclosure:** We will work with you to understand the scope of the issue and provide a fix before disclosing it publicly.

Thank you for helping keep Pixel Pong secure!
