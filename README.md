# Pixel Pong 🕹️

**Pixel Pong** is a premium, arcade-style reimagining of the classic Pong game, built with Python and Pygame-CE. It features a complete visual and audio overhaul with vibrant gradients, glassmorphism UI, particle systems, and dynamic gameplay mechanics.

Built by **Subhan Haider**.

## ✨ Features
- **Arcade Masterpiece UI:** Stunning atmospheric gradients, neon glow effects, and interactive electric buttons with particle trails.
- **Multiple Game Modes:** 
  - **Single Player:** Face off against an adaptive AI.
  - **Local Versus:** Grab a friend and play on the same keyboard.
  - **Chaos Mode:** Two balls at once for maximum intensity!
- **Dynamic Themes:** Instantly switch between curated color palettes (Neon, Emerald, Sunset) that change the entire look of the game.
- **Super Attack:** Build up your super meter and press `Left Shift` to unleash a charged, high-speed attack!
- **Explosive Bricks:** Hit bricks for extra points, and watch out for the exploding ones that shake the screen.
- **Universal Controls:** Play comfortably using `W/S`, `Up/Down Arrows`, or `I/K`.

## 🛠️ Requirements
- Python 3.10+
- `pygame-ce` (Pygame Community Edition)

To install the required dependencies, run:
```bash
pip install pygame-ce
```

## 🚀 How to Run
Navigate to the project directory and run the main script:
```bash
python main.py
```

## 🎮 Controls
- **Player 1:** `W` / `S`
- **Player 2:** `Up` / `Down` arrows or `I` / `K`
- **Super Attack:** `Left Shift` (when meter is full)
- **Fullscreen Toggle:** `F`
